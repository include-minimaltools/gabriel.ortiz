Haceme un paro

  • Author: include-minimaltools
  • Fecha:8/3/2023

Registro de Microprestamos entre Amigos

“Haceme un paro” es una aplicación móvil diseñada para llevar un registro de microprestamos entre amigos. Su objetivo es facilitar el seguimiento de préstamos pequeños realizados entre los usuarios. La aplicación permite registrar manualmente los préstamos, enviar y aceptar solicitudes de amistad, y llevar un registro de las deudas pendientes entre los amigos. Utiliza Firebase para gestionar la autenticación de usuarios, almacenar datos en tiempo real y enviar notificaciones push mediante Firebase Messaging.

Funcionalidades

  • Inicio de sesión con Google: La aplicación cuenta con un sistema de autenticación mediante Google, lo que permite a los usuarios guardar sus transacciones en la nube y vincular sus cuentas en diferentes dispositivos. La autenticación con Google garantiza la seguridad y confidencialidad de la información de los usuarios.

  • Solicitudes de amistad: Una vez que los usuarios han iniciado sesión, tienen la opción de enviar solicitudes de amistad a otros usuarios dentro de la aplicación. Al ser aceptadas, los amigos aparecerán en una lista que facilita el proceso de realizar préstamos entre ellos.

  • Registro de préstamos: Para realizar un préstamo, el usuario debe ingresar el monto prestado y una descripción del mismo. Al hacerlo, la aplicación enviará una notificación al teléfono del amigo al que se le está realizando el préstamo. El amigo debe aceptar la solicitud para que el préstamo quede registrado en la aplicación.

  • Registro de deudas y cuentas pendientes: La aplicación mantiene un registro de los préstamos realizados y las deudas pendientes entre amigos. De esta manera, los usuarios pueden verificar fácilmente cuánto dinero le deben a cada amigo y cuánto dinero deben recibir de ellos. Esto permite mantener una cuenta clara y precisa de las transacciones entre los usuarios.

  • Cierre de cuentas: Cuando los usuarios deciden saldar las deudas, la aplicación facilita el cierre de cuentas, indicando cuál es el monto final a pagar o recibir. Los usuarios pueden reunirse y pasarse el dinero tanto en efectivo como por transacción para resolver las deudas pendientes.

Tecnologías

La aplicación “Haceme un paro” se desarrolló utilizando las siguientes tecnologías:

  • React Native: React Native es un framework de desarrollo de aplicaciones móviles que permite construir aplicaciones nativas para iOS y Android utilizando JavaScript. Esta tecnología proporciona una experiencia de usuario fluida y una interfaz de usuario atractiva.

  • Firebase Authentication: Firebase Authentication es un servicio de Firebase que ofrece soluciones de autenticación seguras y confiables. En “Haceme un paro”, se utiliza para permitir el inicio de sesión con Google y gestionar la identidad de los usuarios.

  • Firestore: Firestore es una base de datos en tiempo real de Firebase que permite almacenar y sincronizar datos entre los usuarios de manera eficiente. En la aplicación, se utiliza para almacenar los registros de préstamos y las cuentas pendientes.

  • Firebase Storage: Firebase Storage es un servicio de Firebase que proporciona almacenamiento de archivos en la nube. En “Haceme un paro”, se utiliza para guardar y gestionar archivos relacionados con la aplicación.

  • Firebase Messaging: Firebase Messaging es un servicio de Firebase que permite enviar notificaciones push a los dispositivos móviles de los usuarios. Esta tecnología se emplea en la aplicación para enviar notificaciones al amigo receptor del préstamo y solicitar su aceptación.
