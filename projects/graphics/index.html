Game Travel Offline App

Game Travel Offline es un proyecto experimental desarrollado para la clase de Programación Gráfica. El juego ofrece una experiencia de juego única utilizando gráficos primitivos dibujados línea por línea. El objetivo principal de este proyecto es explorar y aplicar conceptos de programación gráfica, así como aprender a utilizar herramientas como Canvas en Kotlin para crear un juego interactivo.

¿Qué es?

Game Travel Offline es un juego offline de aventura y exploración diseñado como parte de un proyecto experimental para la clase de Programación Gráfica. El enfoque principal del proyecto es experimentar con la programación gráfica y aprender a crear gráficos primitivos dibujados línea por línea utilizando la herramienta Canvas en Kotlin.

Funcionalidades

Dado que Game Travel Offline es un proyecto experimental, las funcionalidades pueden ser limitadas y se centrarán en la exploración y experimentación de conceptos de programación gráfica:

  1. Exploración del Mundo Virtual: Los jugadores podrán explorar un mundo virtual creado con gráficos primitivos dibujados con la herramienta Canvas en Kotlin.

  2. Interacción con el Entorno: Aunque los gráficos pueden ser primitivos, se buscará implementar interacción con elementos del entorno y la posibilidad de realizar acciones básicas.

  3. Experiencia de Aprendizaje: El proyecto se enfocará en proporcionar una experiencia de aprendizaje, donde los desarrolladores podrán experimentar con la programación gráfica, aprender a dibujar gráficos línea por línea y aplicar conceptos aprendidos en la clase de Programación Gráfica.

Tecnologías Utilizadas

El desarrollo de Game Travel Offline se llevará a cabo utilizando las siguientes tecnologías:

  • Kotlin: Kotlin es un lenguaje de programación versátil y moderno que se utilizará para desarrollar el juego y experimentar con conceptos de programación gráfica.

  • Canvas en Kotlin: La herramienta Canvas en Kotlin será utilizada para dibujar gráficos primitivos línea por línea. Con esta tecnología, se podrán experimentar y aprender diferentes técnicas de dibujo y diseño gráfico.

Game Travel Offline es un emocionante proyecto experimental que brinda la oportunidad de sumergirse en la programación gráfica y explorar la creación de gráficos primitivos utilizando la herramienta Canvas en Kotlin. A través de este proyecto, los desarrolladores tendrán la posibilidad de aprender y aplicar conceptos clave de programación gráfica mientras crean una experiencia de juego única y entretenida.
