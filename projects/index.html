Inwent System

El Sistema de Facturación e Inventario es una aplicación desarrollada con tecnologías modernas que permiten una gestión eficiente de las operaciones comerciales y de inventario del negocio

Haceme un paro

'Haceme un paro' es una aplicación móvil diseñada para llevar un registro de microprestamos entre amigos. Su objetivo es facilitar el seguimiento de préstamos pequeños realizados entre los usuarios.

Akdmik

Akdmik es un sistema web diseñado para el registro de notas del colegio Liceo Cultural Nicaragüense. Ofrece una plataforma completa para administrar y gestionar todas las áreas relacionadas con las calificaciones y el desempeño académico de los estudiantes.

Aplicación Ley 431 de Nicaragua

La aplicación 'Ley 431 de Nicaragua' proporciona un recurso educativo y de estudio para aquellas personas que necesitan tramitar una licencia de conducir en Nicaragua.

Administrador de Ley 431

Game Travel Offline App

Game Travel Offline es un proyecto experimental desarrollado para la clase de Programación Gráfica. El juego ofrece una experiencia de juego única utilizando gráficos primitivos dibujados línea por línea.

Visions

Visions es una aplicación desarrollada en Unity que tiene como objetivo dinamizar el estudio convencional al incorporar herramientas interactivas y modelos 3D en el proceso de aprendizaje.

EVA Lightning

Eva Lightning es un proyecto de rediseño de la interfaz del Entorno Virtual de Aprendizaje utilizado por la Universidad Nacional de Ingeniería.

Inquiry Log

Inquiry Log es un sistema web desarrollado específicamente para los maestros de la Universidad Nacional de Ingeniería.

AED Hub

AED Hub es una recopilación completa de todas las actividades relacionadas con algoritmización y estructuras de datos
