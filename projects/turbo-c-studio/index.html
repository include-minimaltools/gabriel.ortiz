Turbo C Studio

  • Author: include-minimaltools
  • Fecha:8/3/2021

Debido a que la Universidad Nacional de Ingeniería (UNI) tiene como principal lenguaje para la enseñanza de lógica de programación el lenguaje de Turbo C 2.0, sus estudiantes deben presentar proyectos y aplicaciones codificadas desde el emulador de DOSBox, esto se puede profundizar más a detalle en el artículo Turbo C 2.0. Esto supone muchas limitantes a la hora de desarrollar una aplicación y debido a esos problemas surge la aplicación de Turbo C Studio

¿Qué es Turbo C Studio?

Turbo C Studio (TCS) es un asistente para la creación de formularios y un entorno virtual gráfico en MS-DOS mediante el cual se pretende simular una versión reciente del sistema operativo de Windows 11, también funciona como instalador de la máquina virtual que simula MS-DOS “DOSBOX” y del lenguaje de programación utilizado ‘Turbo C 2.0’, además anexa la funcionalidad de instalar un editor de texto de Microsoft llamado Visual Studio Code.

Abre tus programas fácilmente

Turbo C Studio trabaja en conjunto a los programas antes mencionado para facilitar la programación en el lenguaje de Turbo C 2.0, aunque se sabe de que seguirá siendo aparatosa. TCS abrirá diversos programas puestos en tu carpeta de BIN directamente en DOSBOX sin necesidad de estar buscando el archivo dentro del Entorno de Turbo C y te dará la opción de abrir dicho código en Visual Studio Code

Windows C

La principal función de TCS es generar un archivo C (.c) con el cual se pueda ejecutar una simulación básica del sistema operativo Windows 10 con la finalidad de que se puedan anexar programas al sistema con funcionalidades separadas. Los códigos generados por default se irán mejorando con diversas versiones del asistente. Se está trabajando en el proyecto y se subirá la primera versión el 8 de Marzo del 2021

Formularios

Para la creación de formularios, TCS te generará un archivo C (.c) el cual te guardará en la carpeta BIN de tu directorio, este archivo tendrá un código de preestablecido para poder visualizar en el modo gráfico un “Form” parecido al ya visto en Visual Studio con Visual Basic o C#, esto pretende ahorrar tiempo al programador al comenzar a realizar un proyecto y que se centre en las funcionalidades que formarán parte del formulario y no en las virtudes básicas

En dado caso se quiera generar un archivo .h se eliminarán ciertas funcionalidades y se introducirá en la carpeta de “Include”, permitiendo así que en archivo Windows.c se ejecute con una simple función, habiendo claro, incluido la librería.

Configura DOSBox

Otra de las utilidades que presenta Turbo C Studio es el agregar una interfaz con la que puedes configurar el emulador de DOSBox o reestablecerlo en un caso introduzcas un valor incorrecto o el archivo de configuración se corrompa por algún motivo. Esto además de que ayuda a la hora de presentar proyectos con una buena configuración, te permite modificar valores como los ciclos de velocidad para lograr una mayor fluidez sin morir en el intento

Conclusión

Si eres de primer año de la carrera de Ingeniería en Computación en la UNI, esta aplicación será de mucha ayuda para ti y tus iniciaciones en el desactualizado Turbo C 2.0, te servirá para guiarte en el tema de funciones, tipos de datos, estructura de datos, punteros, modo gráfico, etc. Te invitamos cordialmente a utilizar el programa para tu beneficio, sin embargo, el plagio del código dado con el archivo .c de Windows sería muy evidente en caso que lo quieras entregar como tuyo propio, así que te recomendamos que no lo hagas.
